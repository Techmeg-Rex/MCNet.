MCNet is a custom launcher and companion Fabric mod, built and maintained by a single developer for a small, private modded Minecraft community. It's not a commercial product — it exists to make one group of friends' server easier to install, update, and play on.

What it is

MCNet pairs a desktop launcher with a lightweight Fabric mod so that joining, installing, and updating modpacks for the server takes one click instead of a manual folder-by-folder setup. It handles Microsoft sign-in, modpack installation from Modrinth and CurseForge, automatic backups, and switching between the server's different modpacks without leaving the app.

It was built the way most small-community launchers are: because walking friends through manual Java/mod setup got old, and a one-click "just play" experience was worth building.

One-click installs

Fetches and installs the server's modpacks directly from Modrinth / CurseForge, no manual mod management.

Microsoft sign-in

Standard OAuth sign-in through Microsoft/Xbox — MCNet never sees or stores account passwords.

Multi-pack support

Switches between the server's different modpacks (survival, creative, tech-focused, etc.) from one launcher.

Self-contained

Backups, updates, and Java/runtime management are handled locally — no external account or profile system.

02

Scope & who it's for

MCNet is used by a small, closed group of players on one private Minecraft server. It is not distributed publicly, not monetized, and not intended to scale beyond the community it was built for.

For anyone reviewing this as part of a Microsoft API / app registration request:

MCNet uses the standard Microsoft identity OAuth flow purely to let a player's existing Minecraft-linked Microsoft account sign in and launch the game — the same purpose every third-party Minecraft launcher uses it for. No credentials are handled or stored by MCNet directly; authentication is delegated entirely to Microsoft. The app has a small, fixed user base and no commercial component.
